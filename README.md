# react-expense-manager

React Expense Manager

A Simple React Expense Manager to Keep track of your daily expenses.

How to Start Dev Server

1. Clone Repository
2. In the Root `npm install`
3. `cd client && npm install`
4. ``cd ..`
5. `npm run dev`
6. Both The client and Server will launch.
